Two files can be found in folders regarding each round of the experiment: 
- default.xml
- <system>.pdf

The file 'default.xml' is used as an input for the PROPHET tool and the pdf is the feature model for the system under evaluation.

:: The PROPHET tool is available at: https://github.com/feigensp/Prophet/
:: ALGORITHMS.JS is available at: https://github.com/felipernb/algorithms.js
:: VIDEO.JS is available at: https://github.com/videojs/video.js
:: ALGORITHMS.JS-SPL is available at: https://github.com/alcemirsantos/algorithms.js-spl
:: VIDEO.JS-SPL is available at: https://github.com/alcemirsantos/video.js-spl
